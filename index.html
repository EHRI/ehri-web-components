EHRI Web Component Demo

This library provides one custom tag: <ehri-item>.

EHRI Portal Items

The <ehri-item> tag displays a panel providing info about an item in the EHRI Portal. It requires a single attribute named item-id containing the global identifier of the entity.

The following types of item are currently supported:

  • Archival Institutions
  • Archival Descriptions
  • Archival Authorities
  • Country Reports

Example code:

    <script src="ehri-web-components.js"></script>

    <ehri-item item-id="us-005578"></ehri-item>

Examples:

Archival Institutions

<ehri-item item-id="us-005578"></ehri-item>

Archival Descriptions

<ehri-item item-id="il-002798-4019665"></ehri-item>

Archival Authorities

<ehri-item item-id="ehri_cb-001658"></ehri-item>

Country Reports

<ehri-item item-id="de"></ehri-item>
