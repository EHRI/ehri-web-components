EHRI Web Component Demo

This library provides two custom tags: <ehri-item>, and <ehri-resource>.

EHRI Portal Items

The <ehri-item> tag displays a panel providing info about an item in the EHRI Portal. It requires a single attribute named item-id containing the global identifier of the entity.

The following types of item are currently supported:

  • Archival Institutions
  • Archival Descriptions
  • Archival Authorities
  • Country Reports

Example code:

    <script src="ehri-web-components.js"></script>

    <ehri-item item-id="us-005578"></ehri-item>

Examples:

Archival Institutions

<ehri-item item-id="us-005578"></ehri-item>

Archival Descriptions

<ehri-item item-id="il-002798-4019665"></ehri-item>

Archival Authorities

<ehri-item item-id="ehri_cb-001658"></ehri-item>

Country Reports

<ehri-item item-id="de"></ehri-item>

EHRI Resources

The <ehri-resource> tag shows a panel containing info about an EHRI resource, i.e. a website or other programme provided by EHRI, that is registered on the EOSC Portal. It requires a single attribute: resource-id, containing the EOSC identifier of the EHRI resource.

Examples:

The EHRI Portal

<ehri-resource resource-id="ehri.the_ehri_portal"></ehri-resource>

The Begrenzte Flucht Online Edition

<ehri-resource resource-id="ehri.begrenzte_flucht"></ehri-resource>
