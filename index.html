EHRI Web Component Demo

This library provides an <ehri-item> custom tag. It takes a single "item-id" attribute with the global identifier of the EHRI metadata item you want to embed.

The following types of item are currently supported:

  • Archival Institutions
  • Archival Descriptions
  • Archival Authorities

Example code:

    <script src="ehri-web-components.js"></script>

    <ehri-item item-id="us-005578"></ehri-item>
